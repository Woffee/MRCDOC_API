	无

	notices[]

            type
            from_uid
            from_username
            from_picture
            file_id
            filename
            create_time
            is_read           是否已读
            
        type:2                对方是否接受了你的好友申请
            type
            from_uid
            from_username
            from_picture
            create_time
            is_accept         是否接受
            is_read           是否已读
