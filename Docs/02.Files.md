## 桌面
	
	GET /desk
	
参数：

	page         可无，默认1
	pagesize     可无，默认10
	
返回值：

	files[]
	
## 文件夹

	GET /folder/{folderid}
	
参数：

	page         可无，默认1
	pagesize     可无，默认10
	
返回值：

	files[]
	
## 文件详情

    GET /files/{fileid}

参数

    无
    
返回值

    无
