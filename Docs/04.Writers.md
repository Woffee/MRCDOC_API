# 文档协作者管理API

## 协作者列表
	
	GET /writers
	
参数：

	file_id
	
返回值：

	writers[]
        file_id
        writer_id
        writer_name
        writer_picture
	
## 添加协作者

	POST /writers
	
参数：

	file_id  
	writer_id        要添加的协作者uid
	
返回值：

    无
    
## 删除协作者

    DELETE /writers
    
参数

	file_id  
	writer_id        要添加的协作者uid
    
返回值
    
    无
